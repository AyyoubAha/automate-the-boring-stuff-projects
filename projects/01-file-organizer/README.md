# Project 01 — File Organizer

## Problem

Downloads folders and project folders get messy quickly. This script organizes files into folders based on file type.

## What It Does

The starter version scans a target folder and sorts files into category folders such as:

- images
- documents
- spreadsheets
- archives
- code
- other

## How To Run

```bash
python file_organizer.py /path/to/folder
```

If no folder is provided, the script explains how to use it.
