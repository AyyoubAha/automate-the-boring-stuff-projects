# Automate the Boring Stuff Projects

A public collection of beginner-friendly Python automation projects inspired by real everyday problems.

This repository is part of my journey from zero experience to practical AI Builder. The goal is to learn Python by building useful scripts, not by only watching tutorials.

## Mission

Use Python to automate repetitive tasks such as:

- organizing files
- renaming files
- cleaning spreadsheets
- extracting text from documents
- generating reports
- working with CSV files
- sending structured outputs
- preparing data for later AI workflows

## Project Standard

Every project should include:

1. Problem description
2. What the script does
3. How to run it
4. Example input and output
5. What I learned
6. Future improvements
